# TablicaOgloszen
A web-based notice board application designed for managing and browsing posts. This project focuses on efficient content delivery and comprehensive action tracking.

## Features
- **Post Management**: Users can create and publish posts to the platform.
- **Paginated Feed**: The home page displays all posts with a built-in pagination system to ensure optimal performance and readability.
- **Activity Logging**: The application includes a logging system that tracks specific user actions, such as creating posts and browsing the feed.
## Technologies
- **Backend**: ASP.NET Core MVC
- **Database**: SQL Server

## Installation and Setup
Clone the repository:
``` bash
git clone https://github.com/Rajfex/TablicaOgloszen.git
```
Open the solution file ```(.sln)``` in Visual Studio.

Update the connection string in appsettings.json.

Run migrations to set up the database:
```Update-Database```

Build and run the application.
